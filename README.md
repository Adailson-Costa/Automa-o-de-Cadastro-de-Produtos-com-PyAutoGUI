# Automa-o-de-Cadastro-de-Produtos-com-PyAutoGUI
Processos (RPA) desenvolvido em Python, cujo objetivo é automatizar o cadastro de produtos em um sistema web a partir de um arquivo CSV.  O script simula ações humanas, como cliques de mouse e digitação de teclado, utilizando a biblioteca PyAutoGUI, e realiza a leitura e manipulação dos dados com Pandas.
